# Template Preview Images

This directory contains static preview images for quote templates.

## Image Requirements
- File naming: `template_0.png`, `template_1.png`, ..., `template_10.png`
- Dimensions: 800x800 pixels (or any square aspect ratio)
- Format: PNG with transparent or solid background
- File size: Keep under 200KB for optimal app performance

## Creating Preview Images
1. Run the app and navigate to the Quotes Image Generator page
2. For each template (0-10):
   - Select the template
   - Take a screenshot of the quote image
   - Crop to square aspect ratio
   - Save as `template_X.png` in this directory

## Benefits
- Faster template tab loading (no widget rendering)
- Reduced memory usage
- Smoother scrolling experience
- Consistent preview appearance

Replace the placeholder files with actual template screenshots to see the previews in the Templates tab.

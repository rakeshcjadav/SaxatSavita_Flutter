# Promotional Graphics & Design Guide

## Design Assets Needed for Marketing

### 1. App Store Assets

#### iOS App Store
- **App Icon**: 1024x1024px (PNG, no alpha channel)
- **Screenshots**:
  - 6.7" iPhone: 1290 x 2796 pixels (or 2796 x 1290 for landscape)
  - 12.9" iPad: 2048 x 2732 pixels (or 2732 x 2048 for landscape)
  - Minimum 3, maximum 10 screenshots per device type
- **App Preview Video** (optional):
  - 15-30 seconds
  - 6.7" iPhone: 886 x 1920 or 1920 x 886
  - Max file size: 500 MB

#### Google Play Store
- **App Icon**: 512x512px (PNG, 32-bit)
- **Feature Graphic**: 1024 x 500 pixels (PNG or JPEG)
- **Screenshots**:
  - Phone: 16:9 aspect ratio
  - 7-inch tablet: 1024 x 600 minimum
  - 10-inch tablet: 1920 x 1200 minimum
  - Minimum 2, maximum 8 per device type
- **Promotional Video** (optional):
  - YouTube link
  - 30 seconds to 2 minutes recommended

## 4. Brand Identity Guidelines

### Color Palette

**Primary Colors:**
```
Brand Orange: #FF6B35 (rgb 255, 107, 53)
- Use for: Primary buttons, headers, icons
- Represents: Energy, spirituality, warmth

Brand Blue: #2E86AB (rgb 46, 134, 171)
- Use for: Secondary elements, links
- Represents: Wisdom, peace, trust

Brand Purple: #7B2CBF (rgb 123, 44, 191)
- Use for: Accents, special features
- Represents: Spirituality, transformation
```

**Secondary Colors:**
```
Deep Orange: #E63946
Amber: #F77F00
Teal: #06AED5
Indigo: #5E60CE
```

**Neutral Colors:**
```
Dark Gray: #2B2D42
Medium Gray: #8D99AE
Light Gray: #EDF2F4
White: #FFFFFF
Off-White: #F8F9FA
```

### Typography

**Primary Font**: 
- NotoSansGujarati (for Gujarati text)
- Inter or Roboto (for English text)

**Font Hierarchy:**
```
H1 (Headlines): 48px, Bold
H2 (Subheadings): 36px, Semi-Bold
H3 (Sections): 24px, Semi-Bold
Body: 16px, Regular
Small Text: 14px, Regular
Button Text: 16px, Medium
```

### Logo Usage

**Clear Space:**
- Minimum clear space: Height of "S" in Sakshat on all sides
- Never place logo on busy backgrounds
- Always maintain aspect ratio

**Minimum Sizes:**
- Digital: 120px width minimum
- Print: 1 inch width minimum

**Don'ts:**
- ❌ Don't rotate the logo
- ❌ Don't change logo colors
- ❌ Don't add effects (shadows, gradients)
- ❌ Don't stretch or skew
- ❌ Don't place on low-contrast backgrounds

## 7. Video Content Scripts

### 15-Second App Preview
```
[0-3s] App icon animation + "Sakshat Savita" title
[3-6s] Quick swipe through reading interface
[6-9s] Show reading plan creation (fast motion)
[9-12s] Bookmark a passage
[12-15s] "Download Free" + app store badges
```

### 30-Second Feature Showcase
```
[0-5s] Hook: "Struggling to build a spiritual reading habit?"
[5-10s] Problem: Show calendar with missed days, frustrated user
[10-20s] Solution: Show app features solving each problem
  - Reading plans for consistency
  - Reminders for accountability
  - Progress tracking for motivation
[20-25s] Transformation: Happy user with streak badge
[25-30s] CTA: "Start your journey free - link in bio"
```

### 60-Second Tutorial: Creating a Reading Plan
```
[0-10s] Introduction: "Let me show you how to create your first reading plan"
[10-20s] Open app, navigate to Reading Plans
[20-35s] Set daily goal (time or kirans)
[35-45s] Choose start date and duration
[45-55s] Save and see beautiful visualization
[55-60s] "Now you're ready! Start reading and track your progress"
```

## 9. Content Creation Checklist

### Before Creating Any Graphic:

✅ **Define Purpose**: What action should viewer take?
✅ **Know Platform**: Correct dimensions for where it'll be posted?
✅ **Brand Consistency**: Using approved colors, fonts, logo?
✅ **Clear Message**: Can someone understand in 3 seconds?
✅ **Mobile-First**: Readable on small screens?
✅ **High Contrast**: Text readable against background?
✅ **Call-to-Action**: Clear next step?
✅ **Quality Check**: High resolution (300 DPI for print, 72+ for web)?
✅ **File Format**: Correct format (PNG for transparency, JPEG for photos)?
✅ **Accessibility**: Alt text prepared? Colorblind-friendly?

## 10. A/B Testing Graphics

### Elements to Test:

**Headlines:**
- Version A: "Build Your Spiritual Reading Habit"
- Version B: "Read Sacred Wisdom Daily"
- Version C: "Transform Your Spiritual Practice"

**Colors:**
- Version A: Orange gradient background
- Version B: Blue gradient background
- Version C: Purple gradient background

**Call-to-Action:**
- Version A: "Download Free"
- Version B: "Start Your Journey"
- Version C: "Join Thousands of Readers"

**Images:**
- Version A: App screenshot
- Version B: Happy user with phone
- Version C: Abstract spiritual imagery

### Testing Methodology:
1. Create 2 versions changing only ONE element
2. Run for 7 days minimum
3. Measure click-through rate
4. Implement winner
5. Test next element

## 14. Quick Reference: Social Post Templates

### Image Dimensions Cheat Sheet

```
Instagram: 1080 x 1080 (square) or 1080 x 1350 (portrait)
Facebook: 1200 x 630
Twitter: 1600 x 900
LinkedIn: 1200 x 627
Pinterest: 1000 x 1500
Story (all platforms): 1080 x 1920
```

### File Naming Convention

```
YYYYMMDD_platform_campaign_variant.ext

Examples:
20251029_instagram_launch_v1.png
20251029_facebook_feature_bookmarks.jpg
20251029_email_header_welcome.png
```

### Export Settings

**For Social Media:**
- Format: PNG (with transparency) or JPEG (photos)
- Resolution: 72 DPI
- Color Mode: RGB
- Quality: High (90%+ for JPEG)

**For Print:**
- Format: PDF or high-res PNG
- Resolution: 300 DPI
- Color Mode: CMYK
- Include bleed (0.125" extra on all sides)

## 15. Budget-Friendly Tips

### Free Tools Stack:
1. **Canva** - All social graphics
2. **Remove.bg** - Background removal
3. **TinyPNG** - Image compression
4. **Unsplash** - Stock photos
5. **Google Fonts** - Web fonts
6. **Coolors** - Color palettes

### Time-Saving Hacks:
- Create templates for recurring post types
- Batch-create content (e.g., month's worth at once)
- Use Canva's resize feature for multi-platform
- Save brand kit in design tool for consistency
- Repurpose successful posts in new formats

### Quality on a Budget:
- Use high-quality app screenshots (free!)
- Leverage user testimonials (free content!)
- Focus on great copy over fancy graphics
- Simple, clean designs often perform best
- Consistency > complexity
